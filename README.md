# 🔐 DualLayer – SOC Analyst & Cybersecurity Labs

Bienvenido a mi laboratorio de ciberseguridad.

Soy DualLayer, apasionado por la ciberseguridad con enfoque en dos capas:

- 🔴 Offensive Security (Pentesting)
- 🔵 Defensive Security (SOC Analyst)

Mi objetivo es entender cómo funcionan los ataques y cómo detectarlos, analizarlos y responder a ellos como lo haría un equipo SOC real.

---

## 🧠 Sobre mí

Actualmente me estoy formando en ciberseguridad a través de plataformas como TryHackMe, donde he completado rutas como:

- PreSecurity
- Security 101
- SOC Analyst Level 1 & 2

He desarrollado un laboratorio personal donde simulo ataques en un entorno controlado y realizo análisis como analista SOC.

---

## 🛠 Tecnologías y herramientas

- Análisis de red: Wireshark, tcpdump  
- Seguridad ofensiva: Kali Linux, Hydra, Nmap  
- Sistemas: Ubuntu Server  
- Firewall: UFW (simulación de control de tráfico)  
- SIEM (simulado): Graylog / ELK / análisis de logs manual  

---

## 🔍 Proyectos SOC

Cada proyecto incluye:

- Simulación del ataque  
- Captura de tráfico (Wireshark)  
- Análisis de logs  
- Uso de MITRE ATT&CK  
- Respuesta a incidentes  
- Informe en español  

---

## 📂 Proyectos incluidos

- SSH Brute Force (MITRE T1110)
- Port Scanning (MITRE T1046)
- HTTP Suspicious Traffic
- Reverse Shell & Beaconing (MITRE T1071)
- Lateral Movement (MITRE T1021)
- Firewall Analysis
- Incident Response Simulation
