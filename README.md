# 🔐 SSH Brute Force Attack

## 📌 Descripción
Simulación de un ataque de fuerza bruta contra un servidor SSH en un entorno controlado de laboratorio.

## 🎯 Objetivo
Detectar intentos de acceso no autorizados mediante múltiples credenciales.

## 🧠 MITRE ATT&CK
- T1110 – Brute Force
- Categoría: Credential Access

## 🛠 Herramientas utilizadas
- Kali Linux
- Hydra
- Wireshark
- Logs del sistema (auth.log)

## 📊 Qué se analiza
- Intentos fallidos de login
- Tráfico repetitivo en puerto 22
- Dirección IP atacante
